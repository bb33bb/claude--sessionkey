claude使用sessionkey一键登陆插件

下载文件解压到指定文件夹a

使用教程
1. 打开谷歌浏览器，在地址栏中输入 `chrome://extensions/` 并按下回车键，进入扩展程序管理页面。
2. 在右上角的开发者模式下勾选复选框，以启用开发者模式。
3. 点击页面左上角的“加载已解压的扩展程序”按钮。
4. 在弹出的对话框中，选择您创建的插件文件夹并点击“指定文件夹a”。

5. 
   然后就可以使用插件了
